#0:代码块的引用

代码块引用的语法是:

(此处有1个空格)```language
code
(此处有1个空格)```

language指的是语言类型，加了的话可以代码高亮（如果支持）。如果不在开头打一个空格，在某些环境下会出意想不到的问题，我也不知道为什么。
